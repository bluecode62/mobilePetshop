# 🐶 펫쇼핑몰 댕냥베베(모바일)
 
## 🛠 사용 기술
- HTML5
- CSS3
- JavaScript

## 📂 페이지 구성
- 메인 페이지
- 상품 리스트 서브 페이지
- 상품 상세 디테일 페이지


## ✨ 주요 기능
1. 카테고리 분류버튼 – 메인 페이지 (index.html)
2. API 환경을 가정한 페이지네이션 기눙 구현 – 서브 페이지 (sub.html)
3. 스크롤 구매버튼 노출 – 디테일 페이지 (detail.html)
